# Gruppen-Avatar Standard-Bild

Hier sollte ein Standard-Avatar für Gruppen platziert werden: `group-avatar-default.png`

**Empfohlene Spezifikationen:**
- Format: PNG mit Transparenz
- Größe: 200x200 Pixel
- Inhalt: Icon einer Personengruppe oder abstraktes Symbol
- Farbe: Neutral (Grau/Blau), passend zum Plugin-Theme

**Alternative:**
Du kannst ein Icon aus der PS Community verwenden oder ein eigenes hochladen.

Temporär wird das Plugin auf dieses Bild verweisen, aber es gibt ein sauberes Fallback in den Shortcodes.
